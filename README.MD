## Về tác giả: hoidanit 
===

Các bước cài đặt: (chế độ development)
1. clone code
2. cài đặt thư viện: npm i
3. Update file .env (nếu cần thiết)
4. Chạy dự án:  npm start | hoặc | npm run dev
